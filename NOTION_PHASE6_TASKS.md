## 이미지 분석 결과 요약

제공하신 스케치에서 확인된 주요 기능:

- **영상 크기 조절**: 플레이어 크기 조정 기능
- **반복하기**: 동영상 반복 재생 기능
- **자막 단어 클릭**: 자막의 단어를 클릭하여 단어장에 추가
- **오늘의 단어장**: 추가된 단어들을 표시하는 섹션
- **Q&A 섹션**: AI 기반 질문/답변 기능
- **대본 다운로드**: 대본을 파일로 다운로드
- **배속/소리 조절**: 재생 컨트롤 UI

이 작업들을 완료하면 스케치에서 제시한 모든 기능이 구현됩니다.
